# Changelog

## [1.1.0] - 2026-04-23

### Added
- 命令面板（Ctrl+Shift+P）支持：仅当当前编辑器为 .md 文件时显示命令，否则不可见

## [1.0.0] - 2026-04-23

### Added
- 支持将 .md 文件转换为 Atlassian Wiki Markup (.wiki)
- 右键菜单快捷入口（资源管理器 & 编辑器）
- 转换完成后自动打开生成的 .wiki 文件

### 转换规则
- 标题 h1~h6
- 有序 / 无序列表（支持嵌套）
- 粗体、斜体、删除线、行内代码、链接、图片
- 围栏代码块（统一输出 {code}，不带语言标识符）
- 表格（首行自动识别为表头）
- 水平线

### Confluence 陷阱处理
- 行内裸花括号 { } 自动转义为 \{ \}
- 行内代码含 + - * { } 时改用 [ ] 包裹，避免格式符号冲突
- 代码块不带语言标识符，避免不支持的语言名报错
- 输出文件统一使用 LF 换行符
